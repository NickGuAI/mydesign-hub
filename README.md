# mydesign-hub

`mydesign-hub` stores published design-system files for the mydesign marketplace.

For all content discovery and contribution workflows, use:

- https://mydesign.gehirn.ai/

## Browse Designs

- Gallery: https://mydesign.gehirn.ai/
- Design detail + download: open any design from the gallery

## Contribute a Design

- Upload flow: https://mydesign.gehirn.ai/upload
- Sign in, submit your design package, and the platform will create a review PR into this repo

## Repository Role

- This repo is the content store consumed by mydesign
- Published designs live under `designs/`
- `designs/index.json` is the catalog used by the app
